# TruthLens AI - Fake News Detector

A next-generation AI-powered Fake News Detection web application built using Python, Flask, and Scikit-Learn.
This project uses a highly accurate `PassiveAggressiveClassifier` combined with a robust TF-IDF Vectorizer with N-Gram support to verify article authenticity, served on a beautiful, modern layout featuring glassmorphism elements.

## Features

- **High-Accuracy AI Engine**: Real-time content classification utilizing a `PassiveAggressiveClassifier`.
- **Advanced Text Preprocessing**: Automated URL stripping, noise removal, and bi-gram processing.
- **Flask REST API**: A lightweight backend that persists model states in-memory (`.joblib` dumps) for instantaneous predictions.
- **Premium Frontend Aesthetics**: Dynamic CSS/JS architecture featuring smooth CSS animations, progress tracking, and contextual confidence metrics without reloads.

## Installation and Quick Start

1. **Clone the repository:**
   ```bash
   git clone <your-github-repo-url>
   cd "ibm project"
   ```

2. **Create and activate a virtual environment:**
   - **Windows:**
     ```bash
     python -m venv venv
     venv\Scripts\activate
     ```
   - **Mac/Linux:**
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

3. **Install the dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Train the initial AI Model:**
   *Requires `fake_or_real_news.csv` to be in the parent directory.*
   ```bash
   python train_model.py
   ```
   *Note: Ensure the dataset exists locally as we do not check large files or .joblib artifacts into source control.*

5. **Start the Web Application:**
   ```bash
   python app.py
   ```
   **Open [http://127.0.0.1:5000](http://127.0.0.1:5000) in your web browser to access TruthLens AI.**
